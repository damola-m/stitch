Available Tools

Claude automatically selects the best tool for the job — no instruction needed. You can also ask Claude directly, e.g. "search_web for…" or "search the knowledge base for…"

generate_content Gemini 2.5 Flash

General-purpose multimodal generation. Send text, images, audio, or video files and get a Gemini response. Use this as the default tool for any creative, analytical, or conversational task.

pdf_to_markdown Gemini 2.5 Pro

Converts a local PDF to clean, structured Markdown. Pro-class vision and spatial reasoning gives faithful extraction from scanned documents, mixed-layout reports, and complex specs — including tables.

search_web Gemini 2.0 Flash

Answers questions using live Google Search grounding. Returns cited sources alongside the response. Use for anything requiring up-to-date information that Claude's training data may not cover.

analyse_codebase Gemini 2.5 Pro

Recursively reads a local directory, packs all source files into a single context, and delegates to Gemini's 1M-token window for deep analysis. Use for architecture reviews, security audits, or whole-repo comprehension.

log_solution Local · SQLite

Saves a resolved problem and its solution to your persistent knowledge base. Claude will ask for your confirmation before saving — nothing is stored without your approval. You can also feed it knowledge manually at any time.

search_knowledge SQLite + Gemini 2.5 Flash

Searches your knowledge base before attempting to solve a problem. SQLite FTS5 does a fast keyword pass to find candidates; Gemini then ranks them semantically so the most relevant solution is surfaced — even across thousands of entries.

update_knowledge Local · SQLite

Updates an existing knowledge base entry by ID. Use this to correct a solution, add missing code snippets, or mark an entry as outdated when a better approach has been found.
